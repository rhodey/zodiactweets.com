# zodiactweets.com

I made this site to investigate [RTB Ad Networks](https://en.wikipedia.org/wiki/Real-time_bidding) and practice React.js, now it's just published and kept alive for fun.

## Setup
```
$ npm install
```

## Build
```
$ npm run bundle
```

## Testing
```
$ npm run dev
```
